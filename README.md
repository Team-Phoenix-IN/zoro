# Zoro V3

A powerful, multi-purpose Discord bot built with discord.js v14. Zoro V3 provides a wide range of features to manage your server, entertain your community, and play high-quality music.

[![Add to Discord](https://img.shields.io/badge/Add_Zoro_V3_to_Discord-7289DA?style=for-the-badge&logo=discord&logoColor=white)](https://discord.com/api/oauth2/authorize?client_id=1146290712602759178&permissions=8&scope=bot%20applications.commands)


## ✨ Features

### 🎵 Advanced Music Player
- High-quality audio playback using `Distube`.
- Support for YouTube, SoundCloud, and Spotify links.
- Queue management, looping, volume control, and audio filters.

### 🛡️ Moderation & Server Administration
- Keep your server safe with a robust set of moderation commands.
- Manage roles, kick/ban members, and configure server settings.
- Built-in logging and automated moderation tools.

### 💰 Economy System
- A global/server-wide economy system powered by MongoDB.
- Daily rewards, coin flipping, balance checking, and more.

### 🎲 Fun & Mini-Games
- Engage your community with fun commands and games.
- Interactive commands, meme generation, and games to keep the chat active.

### 🛠️ Utility Commands
- Useful tools like fetching weather data, checking server/user info.
- Fast and reliable API integrations for quick data retrieval.

### 🌐 Web Dashboard (Included)
- Manage your bot settings directly from a web interface!
- Built using Express and EJS with Discord OAuth2 for easy login.
- View bot statistics and manage configurations effortlessly.

## 🚀 Tech Stack

- **Node.js**: `v18+` environment
- **Discord.js**: `v14` for interacting with the Discord API
- **Distube**: Robust music playing framework
- **MongoDB / Mongoose**: Database for storing economy and server settings
- **Express / EJS**: Web framework and templating for the dashboard
- **Passport.js**: For Discord OAuth authentication
